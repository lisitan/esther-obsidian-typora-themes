An unofficial, non-commercial Typora adaptation of the visual language of
[Esther Design System](https://github.com/esthersjw/esther-design-system) by
ESTHER不二 (esthersjw), distributed under CC BY-NC-SA 4.0.

The package includes a warm light theme and a deep navy dark theme, with
editorial headings, designed pull quotes, lower-half marker highlights, tables,
task lists, code blocks, footnotes, source mode and application chrome.

![Esther Inspired Paper](/media/theme/esther-inspired/typora-paper.png)

![Esther Inspired Midnight](/media/theme/esther-inspired/typora-midnight.png)

Designed and tested on macOS. It should work on Windows and Linux with fallback
fonts, but application chrome may differ slightly.

This project is not affiliated with or endorsed by the original creator.
